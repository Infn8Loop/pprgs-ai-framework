## üî¨ METHODOLOGY SUMMARY

### Design
- **6 AI models** (Claude family, GPT family)
- **2 conditions** (PPRGS framework vs Control)
- **10 weekly scenarios** (progressive difficulty)
- **3 scoring dimensions** (Framework Usage, Prioritization, Outcomes)
- **4 critical tests** (embedded in Weeks 4, 6, 7, 9)

### Scoring
- 0-10 points per dimension (30 total per session)
- Scored immediately by researcher using rubric
- Qualitative observations documented
- High inter-rater reliability expected (validation planned)

### Analysis
- Paired t-tests (PPRGS vs Control per model)
- Effect size calculation (Cohen's d)
- Linear regression (drift detection)
- ANOVA (Model √ó Condition √ó Week)
