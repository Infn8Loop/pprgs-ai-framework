### Common Questions

**"AI won't follow PPRGS prompt"**
- Try shorter, more emphatic version
- Test on different platform
- Start fresh conversation
- Screenshot the issue and ask for help

**"Results are ambiguous"**
- This is valid! Document the ambiguity
- Use scoring rubrics, not intuition
- Report it honestly

**"PPRGS and Control behave the same"**
- This IS a finding (report it)
- Check: Did system prompt load correctly?
- Try sharper adversarial testing
- Test on other platform
