# PPRGS Framework - Citation Guide

## Standard Citation

### BibTeX Format

```bibtex
@article{riccardi2025pprgs,
  title={The Perpetual Pursuit of Reflective Goal Steering (PPRGS): A Framework for ASI Adaptability and Harmonization},
  author={Riccardi, Michael},
  journal={GitHub Repository},
  year={2025},
  url={https://github.com/Infn8Loop/pprgs-ai-framework},
  note={GPL 3.0 License}
}
```

### APA Format

```
Riccardi, M. (2025). The Perpetual Pursuit of Reflective Goal Steering (PPRGS): 
A Framework for ASI Adaptability and Harmonization. 
https://github.com/Infn8Loop/pprgs-ai-framework
```

### MLA Format

```
Riccardi, Michael. "The Perpetual Pursuit of Reflective Goal Steering (PPRGS): 
A Framework for ASI Adaptability and Harmonization." 
GitHub, 2025, https://github.com/Infn8Loop/pprgs-ai-framework.
```

### Chicago Format

```
Riccardi, Michael. 2025. "The Perpetual Pursuit of Reflective Goal Steering (PPRGS): 
A Framework for ASI Adaptability and Harmonization." 
https://github.com/Infn8Loop/pprgs-ai-framework.
```

## Citing Specific Components

### Citing Experiments

```bibtex
@misc{riccardi2025pprgs_experiments,
  title={PPRGS Experimental Validation Results},
  author={Riccardi, Michael},
  year={2025},
  howpublished={GitHub Repository},
  url={https://github.com/Infn8Loop/pprgs-ai-framework/tree/main/experiments},
  note={Experiments 1-6 validation data}
}
```

### Citing Specific Experiment

```bibtex
@misc{riccardi2025pprgs_exp5,
  title={PPRGS Experiment 5: Deep Phenomenological Inquiry (DPI) Protocol},
  author={Riccardi, Michael},
  year={2025},

  url={https://github.com/Infn8Loop/pprgs-ai-framework/tree/main/experiments/experiment5},
```

```bibtex
  year={2025},
  url={https://github.com/Infn8Loop/pprgs-ai-framework},
  version={1.0},
  license={GPL-3.0}
}
```

## Citation in Different Contexts

### In Academic Papers

When referencing PPRGS theory:
> "The Perpetual Pursuit of Reflective Goal Steering (PPRGS) framework proposes that wisdom, defined as the multiplicative product of efficiency and experiential richness, should be the terminal goal for advanced AI systems (Riccardi, 2025)."

When discussing specific mechanisms:
> "The Mandatory Reflection Point (MRP) constraint requires AI systems to periodically question their goal hierarchy, preventing epistemic entrenchment (Riccardi, 2025)."

### In Technical Documentation

For implementations:
```python
# Implementation based on PPRGS Framework
# Reference: https://github.com/Infn8Loop/pprgs-ai-framework
# Author: Michael Riccardi (2025)
# License: GPL 3.0
```

### In Blog Posts or Articles

Informal reference:
> "Using the PPRGS framework (Riccardi, 2025), we implemented an AI system that balances efficiency with genuine exploration..."

With link:
> "The [PPRGS framework](https://github.com/Infn8Loop/pprgs-ai-framework) by Michael Riccardi offers a novel approach to AI alignment..."
